## 💡 Як додати новий дизайн до інших сторінок?

### Крок 1: Додати CSS
```html
<link href="modern-style.css" rel="stylesheet">
```

### Крок 2: Використовувати готові класи

**Карточки:**
```html
<div class="card-modern">
    Ваш вміст
</div>
```

**Кнопки:**
```html
<button class="btn-modern">Клік</button>
```

**Текст:**
```html
<h1 class="text-gradient">Заголовок</h1>
```

**Иконки / Аватари:**
```html
<div class="avatar">AB</div>
<div class="avatar large">AB</div>
```

**Помилки:**
```html
<div class="alert-modern alert-danger">Помилка!</div>
```

## 🎬 Анимации в CSS

Готові анімації для використання:

```css
/* Секунду, поки елемент завантажується */
animation: slideUp 0.6s ease-out;
animation: fadeIn 0.6s ease-out;
animation: bounce 2.5s ease-in-out infinite;
animation: float 8s ease-in-out infinite;
```

   - Шукайте помилок в консолі

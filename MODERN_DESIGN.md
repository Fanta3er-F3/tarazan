## 🎬 Ключевые анимации

| Анимация | Описание | Продолжительность |
|----------|---------|-----------------|
| `fadeIn` | Плавное появление | 0.6s |
| `slideUp` | Появление снизу | 0.6s |
| `slideDown` | Появление сверху | 0.6s |
| `bounce` | Прыгающий эффект | 2.5s |
| `float` | Плавающий элемент | 8s |
| `glow` | Светящийся эффект | Бесконечно |
| `shimmer` | Блик через элемент | 3s |
| `shake` | Тряска при ошибке | 0.5s |

## 🔧 Как использовать CSS классы

### Карточки
```html
<!-- Обычная карточка -->
<div class="card-modern">Содержание</div>

<!-- Прозрачная карточка (glassmorphism) -->
<div class="card-glass">Содержание</div>

<!-- Градиентная карточка -->
<div class="card-gradient">Содержание</div>
```

### Кнопки
```html
<!-- Основная кнопка -->
<button class="btn-modern">Клик</button>

<!-- Контурная кнопка -->
<button class="btn-ghost">Отмена</button>

<!-- Маленькая кнопка -->
<button class="btn-modern btn-small">OK</button>

<!-- Большая кнопка -->
<button class="btn-modern btn-large">Отправить</button>
```

### Текст и иконки
```html
<!-- Градиентный текст -->
<h1 class="text-gradient">Заголовок</h1>

<!-- Иконка (avatar) -->
<div class="avatar">JD</div>
<div class="avatar large">JD</div>
<div class="avatar xlarge">JD</div>
```

### Оповещения
```html
<!-- Успех -->
<div class="alert-modern alert-success">Готово!</div>

<!-- Ошибка -->
<div class="alert-modern alert-danger">Ошибка!</div>

<!-- Информация -->
<div class="alert-modern alert-info">Информация</div>
```
